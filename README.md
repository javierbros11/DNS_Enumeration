
# DNS Enumeration

Se trata de un programa que muestra por terminal toda la información que ha sido posible recolectar acerca de un dominio o sitio web almacenada en los diferentes servidores de resolución de nombres (DNS).
## Instalación

A diferencia de otros repositorios, únicamente requerimos de Python en nuestro equipo para poder ejecutar el script.

Instalar Python en caso de que no lo tenga:

[Download Python](https://www.python.org/downloads/)




## Funcionalidades

- **Resolución de nombres**: El script devolverá entre otras cosas la dirección IP asociada a ese nombre de dominio, así como más de una dirección IP en caso de tener más de una.

- **Información "AAAA","CNAME","MX","NS","SOA","TXT"**: La función principal del script es devolver información, principalmente relacionada con direcciones IP ipv4 o ipv6, servicios de correo, etc.

- **Fácil y rápido**: Al ejecutar el script únicamente nos solicitará que introduzcamos un nombre de dominio a buscar. A continuación devolverá toda la información obtenida mostrada en la terminal.
